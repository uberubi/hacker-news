# Hacker News Api Interface

### Demo

https://hacker-news-interface.netlify.app/

### Technologies
  * React
  * React-router-dom
  * Redux
  * Thunk
  * Axios
  * Semantic UI
  * Dayjs (time formatting)
  * NPM
  
### Run

```
git clone https://github.com/uberubi/hacker-news.git
cd hacker-news/
npm install
npm start
```
